¡Bienvenido a la documentación del ejercicio M3C5!
================================

entender ciertos conceptos del lenguaje de programación Python. Cada concepto será explicado de la manera más simple posible
y se irá incrementando gradualmente la complejidad para ver todas las posibilidades. Todo se basará en explicaciones con ejemplos prácticos,
