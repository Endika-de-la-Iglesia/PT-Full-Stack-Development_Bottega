
¡Bienvenido a la documentación de mis ejercicios!
================================

He sido encomendado, como parte de los checkpoints del curso del curso *Full Stack Development with JavaScript, Python, React*, con la tarea 
de generar una documentación sencilla que permita
entender ciertos conceptos relacionados con el desarrollo web. Cada concepto será explicado de la manera más simple posible
y se irá incrementando gradualmente la complejidad para ver todas las posibilidades. Todo se basará en explicaciones con ejemplos prácticos, 
con el fin de ser lo más ilustrativo que se pueda.
